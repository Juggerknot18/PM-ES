Mission

Layered supervision for experimental energy systems

PM-ES investigates how a permanent-magnet electromechanical system can be supervised, protected and observed through a layered architecture. Low-level motor control is delegated to a FOC controller, while an STM32G474 coordinates system states, measurements, permissions and diagnostics. Critical DC-bus protection remains autonomous at the hardware level.

Current status. The supervisory firmware architecture is implemented and has undergone software-level simulation. Hardware integration and bench validation remain in progress. No public performance claim is made until measurements are documented and reproducible.

Telemetry

Public monitoring-interface preview

The Raspberry Pi companion interface is designed to surface system state, DC-bus observation, speed trends, encoder health, communication status, alarms and event history. The public preview uses simulated data and publishes no private dashboard source or implementation detail.

Monitoring only. The public visualization is not connected to hardware and does not expose live control functions, private frame formats, pin assignments or exact protection thresholds.
