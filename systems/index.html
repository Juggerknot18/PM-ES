Systems

Public system overview without sensitive implementation details

The PM-ES public site describes functional roles and development status. It does not publish private hardware release material, pin maps, thresholds or detailed procedures.

Electrical machine

The experimental platform includes a custom permanent-magnet machine. The public site describes its functional place in the architecture while detailed geometry, winding data and measured performance remain subject to separate publication review.

Motor-control interface

A VESC-compatible controller performs field-oriented control. The PM-ES supervisor communicates at a higher level and does not implement phase commutation.

Embedded supervisor

The STM32G474 firmware is divided into modules for brake-chopper monitoring, encoder management, CAN transport, motor supervision and Raspberry Pi telemetry. Responsibility boundaries are intentionally explicit.

Encoder and mechanical reference

Incremental encoder feedback provides position, speed and mechanical reference information used by supervisory logic. Public documentation avoids implementation-specific pin mapping.

DC bus and autonomous brake chopper

The brake chopper is an autonomous hardware protection layer. The STM32 can observe events and perform bounded diagnostics, but it cannot mask or force the hardware protection off.

Raspberry Pi monitoring

A Raspberry Pi dashboard receives status and diagnostic data. In the V1 architecture it is monitoring-only and does not command the motor system.
